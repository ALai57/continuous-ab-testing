Continuous AB Testing

Monitoring an AB test in real time can lead to incorrect decisions. This app allows users to simulate an AB test and see for themselves how errors can arise.

The program simulates an AB test. In the AB test, there are two variations: the original and variation1. You can select the probability of conversion for each variation using the input boxes (e.g. p = 0.6 indicates that there is a 60% chance of success).

The first plot shows the percentage of trials that are successful in both variations. The second plot shows how significant the differences are between the two variations. Typically, p <0.05 is considered significant.
